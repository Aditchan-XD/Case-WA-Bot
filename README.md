# LoL Human Case Rest API WA Bot

# Installation

## Cloning this repo
```cmd
> git clone https://github.com/Paiiss/FummyBOT.git
> cd FummyBOT
```

## Install the package
```cmd
> npm i
```

## Edit config file
Edit the required value in `config.json`. You can get the api key at [`LoL Human Rest API`](http://api.lolhuman.xyz/)
```json
{
    "apikey": "",
    "prefix": "!"
}
```

## Run the bot
```cmd
> npm start
```
