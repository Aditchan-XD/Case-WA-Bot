# LoL Human Case Rest API WA Bot

# Installation

## Termux
```cmd
> pkg update && pkg upgrade
> pkg install git -y
> pkg install nodejs -y
> pkg install ffmpeg -y
```

## Windows
* [`Download ffmpeg`](https://ffmpeg.org/download.html#build-windows)
* [`Download Node JS`](https://nodejs.org/en/download/)

## Cloning this repo
```cmd
> git clone https://github.com/LoL-Human/Case-WA-Bot.git
> cd Case-WA-Bot
```

## Install the package
```cmd
> npm i
```

## Edit config file
Edit the required value in `config.json`. You can get the api key at [`LoL Human Rest API`](http://api.lolhuman.xyz/).
```json
{
    "apikey": "",
    "prefix": "!"
}
```

## Run the bot
```cmd
> npm start
```
